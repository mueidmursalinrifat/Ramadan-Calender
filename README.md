# Ramadan-Calender

```markdown
# রমজান ইফতার টাইম API

Developer : Mueid Mursalin Rifat
Facebook : https://www.facebook.com/mueid.mursalin.rifat1

A simple Flask web app showing Seheri & Iftar times for Bangladesh.

## Features
- All 64 districts with Ramadan calendar
- Seheri & Iftar times
- Countdown timer
- Random duas

## Deploy on Render
1. Push code to GitHub
2. Connect repo at [render.com](https://render.com)
3. Set:
   - Build: `pip install -r requirements.txt`
   - Start: `python app.py`
4. Deploy

## Deploy on Vercel
1. Push to GitHub
2. Import at [vercel.com](https://vercel.com)
3. Auto-detects Python
4. Deploy

## Local Setup
```bash
pip install -r requirements.txt
python app.py
# Open http://localhost:5000

Files

· app.py - Flask backend
· templates/index.html - Frontend
· requirements.txt - Dependencies
